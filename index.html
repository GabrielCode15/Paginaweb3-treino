Anatomia em HTML

HTML (HyperText Markup Language)

  • é uma linguagem de marcação que define a estrutura e a apresentação do texto bruto.
  • em HTML, o computador pode interpretar o texto bruto que é agrupado em elementos html.
  • hipertexto é um texto exibido em um computador ou dispositivo que fornece acesso a outro texto por meio de links, também conhecidos como hiperlinks.

O html é composto de elementos. Esses conhecimentos estruturam a página da web e definem o seu conteúdo.

Um elemento é composto por:

  • uma tag de abertura (<p>);
  • o conteúdo (Hello world !);
  • uma tag de fechamento (</p>);

Uma tag e o conteúdo entre ela são chamados de elemento HTML.
Existem muitas tags que podemos usar para organizar e exibir texto e outros tipos de conteúdo, como imagens.

O HTML é um conjunto de tags responsáveis pela marcação de conteúdo de uma página no navegador.
As tags são os elementos a mais que escrevemos usando a sintaxe <nome da tag>.
Diversas tags são disponibilizadas pela linguagem html e cada uma possuí uma funcionalidade específica.

Estrutura Básica de um documento HTML

Ao criar um novo documento html, devemos sempre escrever a estrutura básica de um documento desse formato.
A estrutura básica de um documento em html é composto por 11 linhas.

  1. <!DocType htmml>
  2. <html long= en>
  3. <head>
    1. <meta charset= UTF-8>
    2. <meta name=Viewport content= Width= device-Width, initial-scale= 1.0>
    3. <title>Document<title>
  4. <head≫
  5. <body>
  7. </body>
  8. </html>
Vejamos então, o significado de cada linha da estrutura:
Linha 1
Indica que o documento atual será escrito na versáo mais atualizada da linguagem (no casom html5).
Linha 2 e 11
Delimitam o documento html, que é sempre dividido em duas partes: a cabeça e o corpo. Na linha 2, também estamos a indicar que o conteúdo desse site será no idioma inglês.
Linha 3 e 7
Delimitam a cabeça da página, local onde saõ realizadas algumas configurações iniciais como formatos, estilos, ícones de favoritos, etc.
Linha 4
Adiciona ao documento atual o suporte a caracteres acentuados.
Remover esta linha pode causar erros na renderização de algumas letras na tela.
Linha 5
Indica que o conteúdo aparecerá, por padráo, ocupando todo o espaço disponível da tela e com uma escala de 1:1.
Linha 6
Configura a o título da página.
Linha 8 e 10
Delimitam o corpo da página, a maior porção do site, que vai aparecer na tela. É aqui onde colocaremos todo o nosso conteúdo.

Titulos

Os títulos em html são semelhantes aos títulos em outros tipos de mídia. Por exemplo, em jornais, títulos grandes são normalmente usados para captar a atenção do leitor.
Outras vezes, os títulos são usados para descrever o conteúdo, como o título de um filme ou um artigo educacional.

O html segue um padrão semelhante. Em html, existem 6 títulos diferentes ou elementos de título.
Os títulos podem ser usados para diversos fins, como títulos de secções, artigos ou outras formas de conteúdo.

A seguir está a lista de elementos de cabeçalho disponíveis em html.
Eles são ordenados do maior para o menor em tamanho.

Vejamos a lista:

  1. <h1>

  2. <h2>

  3. <h3>

  4. <h4>

  5. <h5>
  6. <h6>

O conteúdo de um h1 é o maior, e o conteúdo de um h6 é o menor.
O tamanho exato em que cada navegador mostra os títulos podem variar ligeiramente. Os utilizadores também podem ajustar o tamanho do texto no seu browser.

Listas Ordenadas

A lista ordenada é criada com a tag <ol>.
Dentro da tag ol devemos utilizar em vez da tag p que usamos para escrever texto, devemos substítui-la pela tag <li>.

<li>
Cada item da lista está colocado entre uma abertura li e um fechamento /li.
O li significa item da lista.
Vejamos entáo, como se usa as listas ordenadas com este exemplo:

<ol>
<li> Pique as batatas em quartos </li> <li> Ferva em água salgada por 15-20 minutos até ficar macio </li>
<li> Aqueça o leite, manteiga e noz-moscada </li> <li> Escorra as batatas e amasse </li>
<li>Misture na mistura de leite </li> </ol>

Resultado:
  1. Pique as batatas em quartos;
  2. Ferva em água salgada por 15-20 minutos até ficar macio;
  3. Aqueça o leite, manteiga e noz-moscada;
  4. Escorra as batatas e amasse;
  5. Misture na mistura de leite;

Listas não ordenadas

A lista não ordenada é criada com a tag ul.
O processo é exatamente igual ao da tag ol, e o uso da tag li, deve continuar da mesma forma.
A única diferença é na tela será um ponto (a cada item) e m vez de um número.

Vejamos um exemplo:

<ul>
<li>Sal e pimenta a gosto</li>
<li>6Kg de farinha de trigo</li>
</ul>

Resultado:
  • Sal e pimenta a gosto;
  • 6 Kg de farinha de trigo;

Lista de Definição

A lista de definição é criada com a tag dl e geralmente consiste de uma série de termos e suas definições.
Dentro da tag dl devemos usar as tags dt e dd.

Dt
É utilizada para conter o termo a ser definido.
Dd
É usada para conter a definição.

Ligações

  • Para criar ligações entre páginas;
  • Ligando para outros sites;

As ligações são a característica definidora da web, porque eleas permitem que você passe de uma página para outra, permitindo assim a ideia de navegaçãp.

Ligações na escrita

As ligações são criadas usando a tag a. Os utilizadores podem clicar em qualquer coisa entre a abertura < a> e o fechamneto </a>. Nós específicamos qual página você deseja vincular o href atribuído.

O texto entre a abertura e o fechamento é conhecido como texto do link.
Sempre que possível, o texto do link deve explicar de onde seráo levados os visitantes.

Para escrever um bom texto de link, você pode pensar em palavras que as pessoas possam usar quando procuram a página que estamos a fazer a ligação.
Por exemplo, em vez de escrever lugares para ficar podemos usar algo mais específico, como hotéis em NEW YORK.

  • Como criar tabelas;

Por exemplo: resultados desportivos, relatórios de ações, horários de comboios.

Estrutura básica de tabela

<table>
O table é usado para criar uma tabela. O conteúdo da tabela sáo escritos linha por linha.
<tr>
Podemos indicar o início de cada linha usando a abertura tr.
(O tr significa linha por linha.)
<td>
Cada célula de uma tabela é representado usando a tag td.
(O <td> significa os dados da tabela.)
<th>
O th ~e usado apenas como o td, mas o seu objetivo é representar o título, uma coluna ou uma linha.
